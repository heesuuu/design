Samsung Electronics, the first store-exclusive icon
This project aimed to create a consistent set of standard icons for Samsung Electronics to intuitively convey our products and key points of differentiation (POD) to customers. We addressed the issue of differing icon styles across media and prioritized creating metaphors that would intuitively convey information about product functions and key features to users.
Starting with TVs and refrigerators, we designed 60 icons ranging from products to key PODs. We also created design guidelines for ongoing management. Currently, Samsung Electronics is conducting marketing using these icons in stores nationwide.
